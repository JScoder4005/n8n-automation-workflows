# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2025-12-05

### Added
- Stock Market Bot workflow
- Multiple stock tracking (Reliance, TCS, Infosys, HDFC Bank)
- Economic Times RSS feed integration for market news
- Merge node to combine stock data and news
- XML parser for RSS feed processing
- Automated daily reports at 4 PM IST on weekdays

### Changed
- Updated README with stock market bot features
- Improved project structure

## [1.0.0] - 2025-12-01

### Added
- Initial weather bot workflow
- OpenWeatherMap integration
- WhatsApp notification support
- Daily weather updates for Bengaluru
